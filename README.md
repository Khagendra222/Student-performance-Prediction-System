## Student Performance Prediction

## Description
This project is a Flask-based web application that predicts student performance using a trained machine learning model. Users enter student details through a web form and receive a PASS or FAIL prediction.

## Features
- Student performance prediction from a web form
- Flask-based user interface
- Trained model loaded from the models folder
- Result page for prediction output

## Technologies Used
- Python
- Flask
- scikit-learn
- joblib
- HTML
- CSS
